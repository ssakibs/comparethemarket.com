## Installation

 - Clone the repository or download the source code
 - install Node 8.9.3 or higher, NPM version 5.5.1
 - Install the dependencies with

	    `npm install`

 - Run the application with
	 - `node index.js`

 - Run test cases with
	 - `npm run test`
